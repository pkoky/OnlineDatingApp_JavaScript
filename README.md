# Online-Dating-App

## 概要
Vue.jsを使ったプロジェクトです。  
オンラインで人を探してメッセージを送り合うことのできるデモアプリを作成しました。  
ランダムなダミーのユーザー情報を取得できるAPI「[RANDOM-USER-GENERATOR]("https://randomuser.me/)」を使い、ユーザー情報を取得しています。  
ルーティング管理に「vue-router」、状態管理に「vuex」を利用しています。



## 機能
- ユーザー一覧ページ
- ユーザー詳細ページ
- チャットページ
- チャット一覧ページ
